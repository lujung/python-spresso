Welcome to python-spresso's documentation!
==========================================

python-spresso is a framework that aims at making it easy to provide authentication
via `SPRESSO <https://spresso.me/>`_ within an application stack.

Status
======

python-spresso is feature complete, as specified in the `SPRESSO Paper <http://infsec.uni-trier.de/publications/paper/FettKuestersSchmitz-TR-spresso-2015.pdf>`_ .
